# OwO-Player
Qin1s+ MusicPlayer

## Hotkeys
- 1 Mute/Unmute
- 2 Volume up
- 4 Previous song
- 5 Pause/continue playing
- 6 Next song
- 7 Navigation between the different tags (not clear to me exactly)
- 8 Volume down
